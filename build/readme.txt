1) Install xampp from  https://www.apachefriends.org/download.html , choose the appropriate version 

2) place the messenger folder in C:/xampp/htdocs

3) start apache and mysql from xampp control panel

4) import the sql file into your phpmyadmin

5) change your database user pass in db.php

6) start your websocket server by opening a cmd in the server folder ( shift+ right-click) and type " php socketserver.php "

7) type ipconfig in cmd 

8) go to your app by typing in < your ip >/messenger in your browser

9) chat with friends in the same network ( do not forget to use smileys :) )
